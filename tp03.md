# Operadores

- Aritméticos:
    - Binários:  `[+ - * / %]`
    - Unários: `[- ++ --]`
- Relacionais/Comparação: `[> >= < <= == !=]`
- Atribuição: `[= += *= /= %=]`
- Lógicos: `[&& || !]`
- Bitwise: `[& | ~ ^ << >>]`
- Referência: &(endereço), *(pointer).

# Controlo Fluxo

- Expressões Condicionais:
    - if
    - switch
- Expressões Cíclicas:
    - for
    - do while
    - while
